Basic VM metrics.

Description

Basic VM metrics. A "plugin" for basic_metrics to export metrics from riak subsystems.

This is for use in applications that themselves include riak_api or riak_core on the 2.x branch where exometer exports are available; basic_metrics does not depend on these applications.
